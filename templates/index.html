How to Install

After you have the zip folder simply follow these steps!

Move the file to the correct location

Locate the file that was just downloaded, it should be called 'random_loot.zip'.

Make sure that you already have a minecraft world save created

Follow the instructions below for your operating system:

Windows Minecraft Location

Your Minecraft should be under: %APPDATA%\.minecraft\saves\**your-world-name**\datapacks

Drag and drop in the zip file you just generated

Relaunch your Minecraft and break a block and enjoy the randomness!!

MacOS Minecraft Location

Your Minecraft should be under: ~/Library/Application Support/minecraft/saves/**your-world-name**/datapacks

Drag and drop in the zip file you just generated

Relaunch your Minecraft and break a block and enjoy the randomness!!

Linux Minecraft Location

Your Minecraft should be under: ~/.minecraft/saves/**your-world-name**/datapacks

Drag and drop in the zip file you just generated

Relaunch your Minecraft and break a block and enjoy the randomness!!
